🎮 Multiplayer Relay Rush
A fast‑paced 2D multiplayer relay game built with Unity and WebGL. Players race against each other in relay‑style challenges, testing speed, timing, and coordination. This project was developed as part of a client contribution and is showcased here as part of my portfolio.

👉 [Play the Demo: Multiplayer Relay Rush WebGL Build](https://asifahmedakash.github.io/Multiplayer_Relay_Rush_Build/)


✨ Features
Multiplayer Relay System – Compete in real‑time relay races.

Smooth 2D Controls – Responsive player movement and transitions.

WebGL Deployment – Play instantly in the browser.

Optimized UI – Clean menus and scoreboard integration.

Immersive Audio – Sound effects for actions and events.

🛠️ Tech Stack
Engine: Unity (WebGL build)

Languages: C# for game logic, ShaderLab/HLSL for visual effects

Platform: GitHub Pages for hosting the playable demo

🚀 Getting Started
Prerequisites
Unity Hub

Unity Editor version 2022.3.x

Installation
Clone the repository:

bash
git clone https://github.com/AsifAhmedAkash/Multiplayer_Relay_Rush.git
Open the project in Unity Hub.

Press Play in the Unity Editor to start the game.
